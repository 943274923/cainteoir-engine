Cainteoir is the Irish Gaeilge word for speaker.

Cainteoir
Text-to-Speech

Design

When creating a text-to-speech program there are various steps that need to be done, which covers:

  1. Document Processing: reading and writing documents
  2. Text Processing: analysis of the text being spoken and converting it to a phonetic representation
