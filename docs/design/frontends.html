Cainteoir is the Irish Gaeilge word for speaker.

Front Ends

The Cainteoir Text-to-Speech engine provides the core functionality that supports converting text into audio.

A standard Graphical User Interface (GUI) is an interface that provides menus, toolbars and dialogs to provide an enhanced user interaction. The idea is to provide applications that support the following libraries:

  1. Qt4 — Use the Qt4 libraries, but not any of the KDE4 libraries. This is to allow it to run easily on non-KDE platforms.
  2. KDE4 — Use the KDE4 libraries and provides KDE integration.
  3. GTK+ — Use the GTK+ libraries to provide native support for Gnome-based platforms.
  4. (N)Curses — This is a text-based interface that provides a GUI-like interface through a terminal.

A command-line interface allows interacting with the text-to-speech engine through a command-line, terminal shell or script.
