| Transcription | CXS       | Lexical Set | Examples |

## Syllables

| Transcription | CXS | Description |
|---------------|-----|-------------|
| `/ˈ/`         | `'` | A syllable with primary stress. |
| `/ˌ/`         | `,` | A syllable with secondary stress. |
| `/./`         | `.` | A syllable boundary. |
